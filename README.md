## 部署

1. 首先从gitee上拉取代码

```shell
git clone https://gitee.com/doushiwoerzi/vue3-blog.git
```

2. 下载依赖

```shell
npm install 或 yarn install
```

3. 本地调试

```shell
npm run dev 或 yarn dev
```

4. 部署

```shell
npm run build 或 yarn build
```
